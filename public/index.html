😎 Web dev & Emojis 😎

Puoi usare una qualunque emoji nel codice HTML senza alcun problema, per esempio:

<button>👋</button>

Ti permette di creare questo

che ovviamente può anche essere stilato

Possono essere usate anche in Javascript a patto che siano all'interno di una stringa

const message = '👍';
document.querySelector('button').innerHTML = message;

E sbircia il sorgente 👀
